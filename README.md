# Diário de Alterações

## Versão 1.04 — 2026-01-06

### Movimentos
- Cadastro de movimentos em janela modal centralizada.
- Botão "Novo movimento" reposicionado acima dos filtros, com filtros agrupados acima da tabela.
- Filtros ampliados: categoria, conta bancária, tipo de pagamento e bandeira.
- Formulário de cadastro com conta bancária (Itaú, Inter, Santander, Banco do Brasil) e tipo de pagamento (Cartão, PIX, Dinheiro, Débito).
- Confirmação de exclusão com descrição e valor do lançamento.
- Normalização dos tipos de pagamento para manter filtros e exibição consistentes.

### Tabelas
- Fonte das tabelas padronizada em 0,7rem.
- Cabeçalhos congelados e rolagem vertical com barra à direita.
- Altura ajustada para exibir 10 linhas visíveis por tabela.
- Botões da tabela de movimentos: editar em azul e excluir em vermelho.

### Parcelamentos
- Filtros por descrição, mês/ano e status (pendente/quitado).
- Botão para limpar filtros.

### Dados
- Template com 300 registros aleatórios de movimentos carregados no sistema.
